代码 
<template>
  <t-tabs v-model="active" :list.sync="list1">
    <t-tab-pane 
      v-for="item in list1" 
      :name="item.name" 
      :label="item.label" 
      :key="item.name">
      <p>这是第{{item.index}}选项卡</p>
    </t-tab-pane>
  </t-tabs>
</template>

<script>
export default {
  data () {
    return {
      active: 'tab2',
      list1: []
    }
  },
  created () {
    for (let i = 0; i < 20; i++) {
      this.list1.push({
        index: i + 1,
        name: 'tab' + (i + 1),
        label: 'tab' + (i + 1)
      })
    }
  }
}
</script>

# slot

遮罩

通过slot插槽，插入遮罩，可插入到需要的位置
默认情况下，dragEnd触发时，会将当前插入到list的最后一个，如果本来就在list里则与最后一个交互
mask上的dragEnd事件可以自定义，return false则默认的事件不会再触发
